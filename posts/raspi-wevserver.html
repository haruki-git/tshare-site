3.ラズパイをWebサーバ化

投稿日：2025-11-23

IT用語の説明

大まかに使用するもの、出てくるもの

LINUXのコマンド

ディストリビューション

HTML：中身・構造を決める、例：タイトル、文章、画像、リンク など。マークアップ言語

CSS：見た目を決める、文字の色・大きさ、背景色、余白、枠線、レイアウト など。スタイルシート言語

その他、コマンドの意味などは、その都度説明を入れていきます。


ラズパイを最新状態にする


sudo apt update
sudo apt upgrade -y

終わったら念のため再起動：


sudo reboot

再起動後、またRealVNCで接続します。


Apache Webサーバをインストール

ターミナルで：


sudo apt install apache2 -y

インストール後、自動で起動します。状態確認：


systemctl status apache2

active (running) と出ていればOKです。qキーで抜けられます。

ブラウザからアクセス確認（まずは自宅LAN内）


hostname -I

同じWi-FiにつながっているPCやスマホでブラウザを開き、

http://192.168.0.23 （←さっきのIPに置き換え）

にアクセスしてみてください。

Apache2 Debian Default Page みたいなページが表示されれば Webサーバとして稼働OK です。

公開フォルダの場所とトップページ編集

ファイル種類ごとのフォルダをつくる

共通デザイン用の CSS をつくる

「共通デザイン用CSS」＝ サイト全体の見た目ルールを1つのCSSファイルにまとめること

各HTMLからlinkタグでそのCSSを読み込むとデザインが統一される


/* 全体の基本レイアウト設定(ファイル名:style.css) */
body {
  font-family: sans-serif;  /* ゴシック体っぽいフォントにする */
  max-width: 800px;         /* 本文の最大幅を800pxにする */
  margin: 0 auto;           /* 中央寄せ */
  padding: 16px;            /* 画面端との余白 */
  line-height: 1.6;         /* 行間を少し広めにして読みやすく */
}

/* ページ上部のヘッダー部分 */
header {
  border-bottom: 1px solid #ccc;  /* 下に区切りライン */
  margin-bottom: 16px;            /* 本文との間に余白 */
}

/* ヘッダー内のタイトル */
header h1 {
  margin-bottom: 4px;  /* タイトルとナビの間を少し空ける */
}

/* ナビゲーションリンクの見た目 */
nav a {
  margin-right: 8px;      /* リンク同士の間隔 */
  text-decoration: none;  /* 下線を消す */
}

/* 記事ごとの下の余白 */
article {
  margin-bottom: 32px;
}

/* ページ下部のフッター */
footer {
  border-top: 1px solid #ccc;  /* 上に区切りライン */
  margin-top: 32px;            /* 本文との間に余白 */
  padding-top: 8px;            /* ラインと文字の間の余白 */
  font-size: 0.9em;            /* 少し小さい文字サイズ */
  color: #666;                 /* 薄いグレーで控えめに */
}
